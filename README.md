# Aurum & Atelier

Proyecto independiente de catálogo premium para relojes y ropa con carrito y pedido por WhatsApp.

## Estructura
- index.html: estructura principal
- styles.css: estilos visuales premium
- app.js: lógica de catálogo, filtros, favoritos, carrito y WhatsApp
- assets/images: recursos visuales locales

## Cómo usar
1. Abrir index.html en el navegador o servir la carpeta con un servidor local.
2. Explorar productos, agregar al carrito y confirmar por WhatsApp.
